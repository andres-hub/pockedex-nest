# Ejecuatar en desarrollo
1. Clonar repositorio
2. Ejecuatar

```
yarn install
```

3. Tener Nest CLI instalado

```
npm i -g @nestjs/cli
```

4. Levantar la db

```
docker-compose up -d
```

5. Reconstruir la base de datos con la semilla

```
http://localhost:3000/api/seed
```


## Stack usado
* MongoDB
* Nest
